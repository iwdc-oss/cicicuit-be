## Instructions

```bash
git clone https://github.com/iwdc-oss/cicicuit-be.git
```

```bash
cd ciciuit-be
```

```bash
cp .env.example .env
(configure using your own database credentials)
```

```bash
npm install
```

```bash
npx prisma migrate dev
```

```bash
npx prisma generate
```

```bash
npm run dev
```
